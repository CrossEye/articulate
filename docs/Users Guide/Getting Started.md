Getting Started
===============

This guide walks you through your first session with Articulate.


### Opening a Document ###

When you first visit Articulate, you'll see a home page listing all available documents.  Click a document title to open it.

The **Document Overview** page shows all versions (branches) of the document.  Click a version to view its latest revision.


### The Main Interface ###

The main reading view has three areas:

| Area           | Location    | Purpose                                          |
| :------------- | :---------- | :----------------------------------------------- |
| Top bar        | Top         | Navigation breadcrumbs, document title, docs link |
| Tree sidebar   | Left        | Collapsible outline of the document structure     |
| Content area   | Center      | The currently selected node and its children      |


### Navigating the Tree ###

The tree sidebar shows the full document hierarchy.  Click the **▸** arrow to expand a section and see its children.  Click any node's label to navigate to it.

The **breadcrumb bar** above the content area shows your current position in the hierarchy (e.g., `ch2 › s203 › C`).  Click any breadcrumb to jump up to that level.


### Reading Content ###

When you select a node, the content area shows:

1. The **selected node** -- its marker, caption, and body text.
2. **Child nodes** -- previews of all direct children, shown below the selected node.

Cross-references (links to other sections) appear as blue dashed-underline text.  Click a cross-reference to navigate to the referenced section.


### Accessing Documentation ###

Click **Docs** in the top bar to open the built-in documentation viewer.  The docs sidebar lets you browse the Users Guide, Developer Guide, API Reference, and Deployment sections.
